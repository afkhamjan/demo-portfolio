# Personal Portfolio Website

A modern, responsive **personal portfolio website** built using **HTML, CSS, and JavaScript**.  
This project showcases my frontend skills, projects, and contact information in a clean and user-friendly design.

##  Features

- Fully **responsive design** (mobile, tablet, desktop)
- Modern **dark theme UI** with smooth color combinations
- **Animated sections** on scroll
- **Interactive navigation menu** for mobile devices
- Clean and simple layout for easy readability

##  Technologies Used

- **HTML5** – Structure and semantic layout  
- **CSS3** – Styling, animations, and responsiveness  
- **JavaScript (ES6)** – Interactivity and DOM manipulation
